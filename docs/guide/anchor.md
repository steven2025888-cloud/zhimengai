# 主播设置

主播设置用于配置主播的讲解音频、话术音频等资源目录。

## 功能说明

### 什么是主播音频目录？

存放主播讲解、话术、产品介绍等音频文件的文件夹。系统会从这个目录读取音频文件，并在触发关键词时播放。

### 为什么要设置？

- **组织音频** - 按产品/场景分类管理
- **快速访问** - 系统自动读取目录中的音频
- **灵活管理** - 随时添加/删除/修改音频文件

---

## 设置步骤

### 第1步：选择或创建目录

1. 进入 **主播设置** 页面
2. 点击 **选择目录** 按钮
3. 选择一个文件夹（或创建新文件夹）
4. 点击 **确定**

### 第2步：组织音频文件

在选定的目录中创建子文件夹，按照以下结构组织：

```
D:\直播音频\
├── 产品讲解
│   ├── 讲解1.mp3
│   ├── 讲解2.mp3
│   └── 讲解3.mp3
├── 欢迎开场
│   └── 开场.wav
├── 促单转化
│   ├── 转化1.mp3
│   └── 转化2.mp3
└── 互动引导
    └── 引导.mp3
```

### 第3步：保存设置

1. 确认目录路径正确
2. 点击 **保存** 按钮
3. 系统会记住这个目录
4. 下次启动时自动使用

## 音频文件要求

### 支持的格式

- `.mp3` - 最常用，兼容性最好
- `.wav` - 高保真，文件较大
- `.aac` - 苹果格式
- `.m4a` - iTunes格式
- `.flac` - 无损格式
- `.ogg` - 开源格式

### 文件命名建议

**✅ 好的命名**：
```
产品介绍.mp3
功能讲解_详细版.mp3
促单话术_v2.mp3
```

**❌ 不好的命名**：
```
1.mp3
文件.mp3
音频.mp3
```

### 文件大小建议

- **单个文件** - 建议不超过50MB
- **总大小** - 建议不超过1GB
- **时长** - 建议单个音频不超过10分钟

## 与关键词的关联

### 关键词分类与音频文件夹的对应关系

| 关键词分类 | 对应音频文件夹 | 说明 |
|-----------|--------------|------|
| 价格 | `zhubo_audio/价格/` | 触发"价格"关键词时播放 |
| 尺寸 | `zhubo_audio/尺寸/` | 触发"尺寸"关键词时播放 |
| 售后 | `zhubo_audio/售后/` | 触发"售后"关键词时播放 |

### 如何关联？

1. 在 **关键词设置** 中创建分类（例如：`价格`）
2. 在主播音频目录中创建同名文件夹（例如：`zhubo_audio/价格/`）
3. 将音频文件放入该文件夹
4. 系统会自动关联

## 常见问题

### Q: 可以不设置主播音频目录吗？

A: 可以。系统有默认目录 `zhubo_audio/`。如果你不设置，系统会使用默认目录。

### Q: 如何修改已设置的目录？

A: 
1. 进入 **主播设置** 页面
2. 点击 **选择目录** 按钮
3. 选择新的目录
4. 点击 **保存**

### Q: 可以使用网络路径吗？

A: 不建议。建议使用本地路径，以确保稳定性和速度。

### Q: 如何快速访问音频目录？

A: 
1. 进入 **AI工作台**
2. 点击 **📂 打开目录** 按钮
3. 系统会打开应用文件夹
4. 找到 `zhubo_audio` 文件夹

### Q: 音频文件太多，如何快速找到？

A: 
1. 使用文件夹分类（按产品/场景）
2. 使用有意义的文件名
3. 定期整理和删除过期文件

## 最佳实践

### 1. 定期备份

- 定期备份 `zhubo_audio` 文件夹
- 避免音频文件丢失
- 建议每周备份一次

### 2. 版本管理

```
zhubo_audio/
├── 产品讲解_v1/
├── 产品讲解_v2/
└── 产品讲解_v3/（当前使用）
```

### 3. 清理过期文件

- 定期删除不用的音频
- 保持目录整洁
- 减少系统负担

### 4. 命名规范

- 使用中文或英文，避免特殊符号
- 包含版本号或日期
- 长度不超过50个字符

## 故障排查

### 系统找不到音频文件

**可能原因**：
1. 目录路径错误
2. 文件夹名称不匹配
3. 文件被删除

**解决方案**：
1. 检查 **主播设置** 中的目录路径
2. 确认文件夹名称与关键词分类一致
3. 检查文件是否存在

### 音频播放失败

**可能原因**：
1. 音频文件损坏
2. 音频格式不支持
3. 文件权限问题

**解决方案**：
1. 尝试用其他播放器打开文件
2. 转换为MP3或WAV格式
3. 检查文件权限

### 目录设置不生效

**可能原因**：
1. 未点击 **保存** 按钮
2. 系统未重启
3. 文件权限问题

**解决方案**：
1. 确认点击了 **保存** 按钮
2. 重启软件
3. 检查文件夹权限
