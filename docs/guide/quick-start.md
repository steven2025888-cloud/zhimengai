# 快速开始

按照以下4个步骤，3分钟内即可跑通整个系统。

## 第1步：设置主播音频目录（可选）

### 什么是主播音频目录？
存放你的讲解音频、话术音频等文件的文件夹。系统会从这个目录读取音频文件。

### 如何设置？

1. 进入 **主播设置** 页面
2. 选择或创建一个文件夹（例如：`D:\直播音频\产品讲解`）
3. 将你的MP3/WAV文件放入该文件夹
4. 按照"产品/场景"建立子文件夹，便于后续管理

### 目录结构示例

```
D:\直播音频\
├── 产品讲解【软件演示可删除，可拖动排序2】
│   ├── 主播讲解1.wav
│   └── 主播讲解2.wav
├── 欢迎开场【软件演示可删除，可拖动排序1】
│   └── spk_1768978871.wav
├── 促单转化【软件演示可删除，可拖动排序3】
│   └── spk_1768978816.wav
└── 互动引导【软件演示可删除，可拖动排序4】
    └── spk_1769096087.wav
```

**提示**：如果不设置，系统会使用默认目录。你可以跳过此步，后续随时修改。

## 第2步：设置关键词（必做）

### 什么是关键词？
当观众在弹幕中说出某个词时，系统自动触发对应的回复或播放。

### 如何设置？

1. 进入 **关键词设置** 页面
2. 点击 **新建分类**（例如：价格、尺寸、售后、材质）
3. 为每个分类配置：
   - **必含词**：观众必须说这个词才能触发
   - **意图词**：可选，用于提高匹配精度
   - **排除词**：包含这个词就不触发
   - **回复词**：自动回复的文本（可选）

### 配置示例

**分类名**：价格

| 字段 | 内容 | 说明 |
|------|------|------|
| 必含词 | 多少钱、价格、多少 | 观众说这些词就触发 |
| 意图词 | 便宜、贵、优惠 | 增强匹配精度 |
| 排除词 | 不要、别、不 | 包含这些词就不触发 |
| 回复词 | 这款产品现在有优惠... | 自动回复的文本 |

### 批量导入

如果你有现成的关键词配置（JSON格式），可以：
1. 点击 **导入（合并）**
2. 选择JSON文件
3. 系统会自动合并，避免重复

### 保存并生效

配置完成后，点击 **保存并热更新**：
- 配置立即生效，无需重启
- 下次启动也会保留配置

## 第3步：开启自动化控制（必做）

### 进入AI工作台

点击左侧菜单的 **AI工作台**，你会看到一个中央控制面板。

### 主要开关说明

| 开关 | 功能 | 建议 |
|------|------|------|
| 弹幕监听 | 实时监听观众弹幕 | ✅ 必开 |
| 关键词回复 | 触发关键词时自动回复 | ✅ 必开 |
| 助播接话 | 主播讲完后自动接话 | 可选 |
| 报时播放 | 每隔N分钟播放一条语音 | 可选 |
| AI改写回复 | 用AI改写回复文案 | 可选 |
| 公屏轮播 | 定时发送公屏文案 | 可选 |

### 测试音频

在开启各项功能前，建议先点击 **试听** 按钮，确认：
- 音频文件存在
- 音量正常
- 路径正确

### 其他常用按钮

- **🚀 启动系统** - 启动所有自动化功能
- **⏸ 暂停播放** - 临时暂停音频播放
- **📌 插播音频** - 立即插播一条音频
- **🚨 急插音频** - 优先级最高的插播
- **📂 打开目录** - 快速打开配置文件夹

## 第4步：启动系统 → 扫码登录 → 开始直播（必做）

### 启动系统

1. 在 **AI工作台** 点击 **🚀 启动系统**
2. 系统会弹出登录窗口

### 扫码登录

根据你要直播的平台选择：

**抖音直播**
- 扫描二维码
- 用手机抖音账号确认
- 登录成功后自动进入直播控制台

**微信视频号直播**
- 扫描二维码
- 用微信确认
- 登录成功后自动进入直播控制台

**同时使用两个平台**
- 系统支持同时登录抖音和视频号
- 弹幕会统一显示在控制台
- 自动回复会同时发送到两个平台

### 进入直播控制台

登录成功后，你会看到：
- **实时弹幕** - 观众的评论
- **触发记录** - 哪些关键词被触发了
- **播放队列** - 当前播放的音频
- **快速操作** - 插播、急插、下一条等

### 开始直播

1. 打开你的直播间（抖音/视频号）
2. 开始讲解和互动
3. 观众说出关键词 → 系统自动回复/播放
4. 你专注于讲解和促单

## 常见问题

### Q: 启动后没有反应？
A: 检查以下几点：
- 网络连接是否正常
- 是否成功扫码登录
- 查看日志是否有错误信息

### Q: 关键词没有触发？
A: 可能原因：
- 关键词配置有误（检查必含词）
- 弹幕监听开关未打开
- 观众的弹幕不完全匹配（可调整排除词）

### Q: 音频播放没有声音？
A: 检查以下几点：
- 音频文件是否存在
- 系统音量是否打开
- 音频格式是否支持（MP3/WAV优先）
- 点击"试听"按钮测试

### Q: 可以同时在两个平台直播吗？
A: 可以。系统支持同时登录抖音和视频号，弹幕会统一显示。

### Q: 如何修改已保存的配置？
A: 
- 关键词配置：进入 **关键词设置** 修改后点 **保存并热更新**
- 其他配置：修改后点 **保存** 按钮
- 大部分配置修改后立即生效，无需重启
