关键词匹配规则说明

一、什么是「必含词」？

必含词 = 这个问题在问“谁 / 什么东西”。 只要用户的话里没有出现必含词，这一类问题就不会触发。
例子：分类「炉膛」 必含词：炉膛、内胆
用户弹幕： ① 炉膛多大？ → 命中（包含“炉膛”）
② 这个能装多少？ → 不命中（没说炉膛，也可能在问锅、灰、风机）

二、什么是「意图词」？

意图词 = 用户想问什么类型的问题（尺寸？材质？用法？价格？）。 用于区分：同一个对象，不同问题。
分类：尺寸
必含词：炉膛
意图词：多大、尺寸、宽、高、多少
用户弹幕： ① 炉膛多大？ → 命中必含词+意图，优先匹配（必含词+意图都齐）
② 炉膛怎么样？ → 只命中必含词只有必含词，没有明确意图 → 进入“降级匹配”也会命中，但是优先级次于第一种
③ 内胆厚不厚？ → 无法命中（（没有必含词,没有意图词，不会命中）

三、什么是「排除词」？

排除词 = 出现后立刻否定本分类，用来防止误判。
分类：用不用电
必含词：电
排除词：电话、电商、电视
用户弹幕：
① 这个用电吗？ → 命中
② 我电话打不通 → 被排除（虽然有“电”，但其实在说电话）

四、系统真正的匹配流程

第一优先级： 必含词 + 意图词 同时命中 → 精准触发对应讲解音频 如果没有任何分类命中意图词： 系统自动降级为： 只要命中 必含词 就触发 （适配直播间口语、省略式提问）
用户弹幕： “这个炉膛行不行？” 分析： ✔ 包含必含词：炉膛 ✘ 没有明确意图词（不是尺寸、不是材质、不是灰） 结果： → 系统仍然触发「炉膛讲解」音频（降级命中）

五、总结一句话

必含词：你在问谁？ 意图词：你想问什么？ 排除词：哪些情况要强行排除？ 系统规则： 先找“对象+意图”， 找不到意图时，只要对象对，也照样播报。

六、左侧分类与音频文件命名规则

左侧每一个分类名称，就是对应音频文件的“前缀名”。 系统会自动按「前缀 + 数字」的方式随机播放讲解音频。
例如你新建一个分类：挡风圈

那么对应的音频文件应命名为：
挡风圈1.mp3、挡风圈2.mp3、挡风圈3.wav ……

系统工作方式：
当弹幕命中“挡风圈”这个分类时， 会在所有以“挡风圈+数字”命名的音频中随机选一条播放。
