# 话术改写

话术改写功能使用AI模型自动改写和优化你的讲解文案，让话术更自然、更有说服力。

## 功能说明

### 什么是话术改写？

将你的讲解文案交给AI模型进行改写，使其：
- 更自然、更口语化
- 更有说服力、更容易促成转化
- 更符合直播风格
- 避免生硬的推销感

### 为什么要用话术改写？

- **提升质量** - AI改写让文案更专业
- **节省时间** - 无需手动打磨每条文案
- **提高转化** - 优化的话术更容易促单
- **多版本** - 可生成多个版本选择

---

## 使用步骤

### 第1步：进入话术改写页面

1. 点击左侧菜单的 **话术改写**
2. 查看改写界面

### 第2步：输入原始文案

1. 在左侧输入框中输入要改写的文案
2. 支持粘贴长文本
3. 可输入多条文案（用换行分隔）

**输入示例**：
```
这个产品原价999，现在只需599，性价比超高，赶紧下单吧
```

### 第3步：选择改写风格

| 风格 | 特点 | 适用场景 |
|------|------|--------|
| 温柔 | 柔和、亲切 | 产品介绍、欢迎语 |
| 热情 | 激情、有力 | 促单、转化 |
| 专业 | 正式、权威 | 讲解、说明 |
| 幽默 | 轻松、有趣 | 互动、引导 |

### 第4步：点击改写

1. 选择改写风格
2. 点击 **改写** 按钮
3. 等待AI生成结果

### 第5步：查看和选择结果

1. 右侧会显示改写后的文案
2. 可生成多个版本
3. 选择最满意的版本
4. 点击 **复制** 按钮复制到剪贴板

## 高级功能

### 批量改写

1. 在输入框中输入多条文案（用换行分隔）
2. 点击 **批量改写** 按钮
3. 系统会逐条改写
4. 可一次性导出所有结果

### 对比查看

1. 改写后会显示原文和改写文
2. 可并排对比
3. 选择最满意的版本

### 导出结果

1. 点击 **导出** 按钮
2. 选择导出格式（TXT、Excel等）
3. 保存到本地

## 常见问题

### Q: 改写后的文案可以直接用吗？

A: 可以，但建议：
1. 先试用一次
2. 根据实际效果调整
3. 保留几个版本轮换使用

### Q: 如何改写效果更好？

A: 
1. 输入清晰、完整的原始文案
2. 选择合适的改写风格
3. 如果不满意，可重新改写

### Q: 可以自定义改写风格吗？

A: 目前支持预设风格。如需自定义，可在改写后手动调整。

### Q: 改写会改变原意吗？

A: AI会保留核心信息，但会改变表述方式。建议检查改写结果是否符合原意。

### Q: 如何快速生成多个版本？

A: 
1. 输入原始文案
2. 选择不同的改写风格
3. 分别点击改写
4. 系统会生成多个版本

## 最佳实践

### 1. 准备高质量的原始文案

**✅ 好的原始文案**：
```
这款产品采用进口材料，经过严格质检，原价999元，现在限时优惠599元，性价比超高，欢迎下单
```

**❌ 不好的原始文案**：
```
便宜
好
优惠
```

### 2. 根据场景选择风格

**欢迎开场** → 温柔风格
**产品讲解** → 专业风格
**促单转化** → 热情风格
**互动引导** → 幽默风格

### 3. 定期更新话术

- 每周生成新的改写版本
- 轮换使用不同版本
- 根据效果调整

### 4. 保存优秀版本

- 将效果好的改写文案保存
- 建立话术库
- 便于后续使用

## 故障排查

### 改写失败

**可能原因**：
1. 网络连接失败
2. API服务异常
3. 输入文本过长

**解决方案**：
1. 检查网络连接
2. 稍后重试
3. 缩短输入文本

### 改写结果不满意

**可能原因**：
1. 原始文案质量差
2. 改写风格不合适
3. AI模型限制

**解决方案**：
1. 改进原始文案
2. 尝试其他风格
3. 手动调整结果

### 无法导出结果

**可能原因**：
1. 磁盘空间不足
2. 文件权限问题
3. 系统异常

**解决方案**：
1. 检查磁盘空间
2. 检查文件权限
3. 重启软件
