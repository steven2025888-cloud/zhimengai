# 公屏轮播

公屏轮播功能可以定时自动发送预设的文案到直播间公屏，提升直播氛围和转化。

## 功能说明

### 什么是公屏轮播？

定时自动发送预设的文案到直播间公屏，形成循环播放效果。

### 为什么要用公屏轮播？

- **提升氛围** - 定时发送文案，保持直播热度
- **提高转化** - 重复强调优惠、促单信息
- **节省时间** - 无需手动发送
- **专业感** - 看起来更像专业直播间

---

## 配置步骤

### 第1步：进入公屏轮播页面

1. 点击左侧菜单的 **公屏轮播**
2. 查看轮播配置界面

### 第2步：添加轮播文案

1. 点击 **添加文案** 按钮
2. 输入要轮播的文案
3. 点击 **确定**

**文案示例**：
```
🎉 限时优惠，原价999，现在只需599，赶紧下单！
👍 感谢各位的关注和支持，欢迎加入粉丝群
💝 新用户首单立减100元，快来体验吧
🚀 今天下单送精美礼品一份，数量有限
```

### 第3步：设置轮播间隔

1. 点击 **⏱ N分钟** 按钮
2. 设置轮播间隔时间（建议2-5分钟）
3. 点击 **确定**

### 第4步：启用轮播

1. 进入 **AI工作台**
2. 打开 **公屏轮播** 开关
3. 系统会按照设置的间隔发送文案

## 文案管理

### 添加文案

1. 点击 **添加文案** 按钮
2. 输入文案内容
3. 点击 **确定**

### 编辑文案

1. 选择要编辑的文案
2. 点击 **编辑** 按钮
3. 修改内容
4. 点击 **保存**

### 删除文案

1. 选择要删除的文案
2. 点击 **删除** 按钮
3. 确认删除

### 排序文案

1. 选择文案
2. 使用 **上移** / **下移** 按钮调整顺序
3. 系统会按照顺序轮播

## 轮播规则

### 轮播方式

| 方式 | 说明 |
|------|------|
| 循环轮播 | 按顺序循环发送 |
| 随机轮播 | 随机选择文案发送 |
| 定时发送 | 在指定时间发送 |

### 轮播间隔

| 间隔 | 适用场景 |
|------|--------|
| 2分钟 | 高热度直播间 |
| 3分钟 | 正常直播间 |
| 5分钟 | 低热度直播间 |
| 10分钟 | 讲解类直播间 |

## 常见问题

### Q: 如何设置轮播间隔？

A: 
1. 点击 **⏱ N分钟** 按钮
2. 输入间隔时间
3. 点击 **确定**

### Q: 可以随时修改轮播文案吗？

A: 可以。修改后点击 **保存** 按钮，下一轮会使用新文案。

### Q: 如何暂停轮播？

A: 
1. 进入 **AI工作台**
2. 关闭 **公屏轮播** 开关

### Q: 可以设置不同时间段的轮播文案吗？

A: 目前不支持。建议手动切换文案或联系技术支持。

### Q: 轮播文案有字数限制吗？

A: 建议不超过100个字，避免被平台限制。

### Q: 如何快速添加多条文案？

A: 
1. 点击 **批量添加** 按钮
2. 输入多条文案（用换行分隔）
3. 点击 **确定**

## 最佳实践

### 1. 准备多条文案

- 准备5-10条不同的文案
- 轮流发送，避免重复感
- 定期更新文案

### 2. 根据直播阶段调整

**开场阶段**：
- 欢迎语、互动引导
- 间隔：5分钟

**讲解阶段**：
- 产品介绍、优惠信息
- 间隔：3-5分钟

**促单阶段**：
- 优惠、促单、库存提醒
- 间隔：2-3分钟

**结束阶段**：
- 感谢语、下次预告
- 间隔：5分钟

### 3. 监控效果

- 观察观众反应
- 根据转化率调整文案
- 定期更新内容

### 4. 避免过度轮播

- 间隔不要太短（避免刷屏）
- 文案不要太长（避免被限制）
- 内容要有价值（避免被屏蔽）

## 故障排查

### 轮播没有发送

**可能原因**：
1. 公屏轮播开关未打开
2. 文案为空
3. 网络连接失败

**解决方案**：
1. 进入 **AI工作台** 打开 **公屏轮播** 开关
2. 检查是否添加了文案
3. 检查网络连接

### 发送失败

**可能原因**：
1. 平台API异常
2. 账户权限问题
3. 文案被平台限制

**解决方案**：
1. 稍后重试
2. 检查账户权限
3. 修改文案内容

### 轮播间隔不准确

**可能原因**：
1. 系统时间不同步
2. 网络延迟
3. 平台限制

**解决方案**：
1. 同步系统时间
2. 检查网络连接
3. 增加间隔时间
