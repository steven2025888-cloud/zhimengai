# 功能总览

欢迎使用**织梦AI直播助手**！这是一款专为直播主播设计的智能自动化工具，帮助你在直播中实现自动回复、助播接话、公屏轮播等功能，让你专注于讲解和促单。

## 核心功能

### 1. **AI工作台** - 中央控制面板
- 启动/停止系统
- 实时日志查看
- 各项自动化开关控制
- 本地弹幕测试
- 插播/急插音频控制

### 2. **关键词设置** - 触发规则配置
- 创建分类（如：价格、尺寸、售后等）
- 配置必含词、意图词、排除词
- 设置自动回复文本
- 支持批量导入/导出

### 3. **主播设置** - 音频目录管理
- 设置主播讲解音频目录
- 按场景/产品组织音频文件
- 支持多种音频格式

### 4. **助播设置** - 自动接话配置
- 配置助播语音库
- 设置接话触发条件
- 支持自定义接话文案

### 5. **音色模型** - 语音合成配置
- 选择TTS语音模型
- 调整语音参数
- 试听效果

### 6. **音频目录工具** - 音频管理
- 批量转换音频格式
- 音频文件整理
- 前缀检查

### 7. **AI回复** - 智能改写
- 配置API Key
- 选择AI模型
- 自动改写回复文案
- 提升转化效果

### 8. **话术改写** - 文案优化
- 一键改写讲解文案
- AI优化转化话术
- 支持批量处理

### 9. **评论管理** - 弹幕管理
- 实时弹幕监听
- 评论分类管理
- 快速回复
- 评论记录查看

### 10. **公屏轮播** - 定时公屏
- 设置轮播文案
- 配置轮播间隔
- 支持多条文案循环

## 支持的音频格式

系统支持以下音频格式：
- `.mp3` - 最常用
- `.wav` - 高保真
- `.aac` - 苹果格式
- `.m4a` - iTunes格式
- `.flac` - 无损格式
- `.ogg` - 开源格式

## 快速开始流程

1. **第1步** - 设置主播音频目录（可选，有默认值）
2. **第2步** - 配置关键词规则（必做）
3. **第3步** - 在AI工作台打开需要的自动化开关（必做）
4. **第4步** - 启动系统 → 扫码登录 → 进入直播控制台（必做）

## 常见问题

**Q: 系统支持哪些直播平台？**
A: 目前支持抖音和微信视频号，可同时登录两个平台。

**Q: 关键词触发有延迟吗？**
A: 系统实时监听弹幕，延迟通常在1-2秒内。

**Q: 可以自定义音频吗？**
A: 完全支持。你可以上传自己的MP3/WAV文件到对应目录。

**Q: 如何保证数据安全？**
A: 所有配置本地保存，不上传用户数据。登录信息仅用于平台连接。
