# 评论管理

评论管理功能帮助你高效地管理和回复直播间的弹幕和评论。

## 功能说明

### 什么是评论管理？

实时显示和管理观众的弹幕、评论，支持快速回复、分类管理等功能。

### 为什么要用评论管理？

- **实时监控** - 及时发现观众问题
- **快速回复** - 提高互动效率
- **分类管理** - 有序处理评论
- **记录保存** - 保留互动历史

---

## 界面说明

### 弹幕列表

显示实时的观众弹幕，包括：
- 观众昵称
- 弹幕内容
- 发送时间
- 互动状态

### 评论分类

| 分类 | 说明 |
|------|------|
| 全部 | 所有评论 |
| 未回复 | 还未回复的评论 |
| 已回复 | 已回复的评论 |
| 重要 | 标记为重要的评论 |
| 垃圾 | 垃圾评论 |

---

## 使用步骤

### 第1步：进入评论管理

1. 点击左侧菜单的 **评论管理**
2. 查看实时弹幕列表

### 第2步：查看评论

1. 在列表中选择要查看的评论
2. 查看完整内容
3. 查看观众信息

### 第3步：快速回复

1. 选择评论
2. 点击 **快速回复** 按钮
3. 输入回复内容
4. 点击 **发送** 按钮

### 第4步：分类管理

1. 选择评论
2. 点击 **标记为重要** 或 **标记为垃圾**
3. 系统会自动分类

### 第5步：查看记录

1. 点击 **记录** 标签
2. 查看历史互动
3. 导出记录

## 快速回复

### 预设回复

系统提供常用的快速回复模板：

| 模板 | 内容 |
|------|------|
| 感谢 | 感谢关注，欢迎下单 |
| 价格 | 这款产品现在有优惠... |
| 售后 | 我们提供完善的售后服务... |
| 邀请 | 欢迎加入我们的粉丝群... |

### 自定义回复

1. 点击 **自定义回复** 按钮
2. 输入回复内容
3. 点击 **发送** 按钮

---

## 评论分类

### 标记为重要

1. 选择评论
2. 点击 **标记为重要** 按钮
3. 评论会被加入"重要"分类
4. 便于后续查看和处理

### 标记为垃圾

1. 选择评论
2. 点击 **标记为垃圾** 按钮
3. 评论会被加入"垃圾"分类
4. 可选择删除或隐藏

---

## 记录查看

### 查看互动历史

1. 点击 **记录** 标签
2. 查看所有历史互动
3. 支持按时间、观众、内容搜索

### 导出记录

1. 点击 **导出** 按钮
2. 选择导出格式（Excel、CSV等）
3. 选择时间范围
4. 点击 **导出** 完成

### 记录统计

系统会自动统计：
- 总评论数
- 已回复数
- 未回复数
- 重要评论数

## 常见问题

### Q: 如何快速找到特定的评论？

A: 
1. 使用搜索框输入关键词
2. 或按观众昵称搜索
3. 或按时间范围筛选

### Q: 可以批量回复吗？

A: 
1. 选择多条评论
2. 点击 **批量回复** 按钮
3. 输入回复内容
4. 点击 **发送**

### Q: 如何删除评论？

A: 
1. 选择评论
2. 点击 **删除** 按钮
3. 确认删除

### Q: 可以编辑已发送的回复吗？

A: 
1. 选择回复
2. 点击 **编辑** 按钮
3. 修改内容
4. 点击 **更新**

### Q: 如何导出评论记录？

A: 
1. 点击 **记录** 标签
2. 点击 **导出** 按钮
3. 选择导出格式和时间范围
4. 点击 **导出** 完成

## 故障排查

### 看不到实时弹幕

**可能原因**：
1. 弹幕监听开关未打开
2. 网络连接失败
3. 系统异常

**解决方案**：
1. 进入 **AI工作台** 打开 **弹幕监听** 开关
2. 检查网络连接
3. 重启软件

### 快速回复失败

**可能原因**：
1. 网络连接失败
2. 平台API异常
3. 账户权限问题

**解决方案**：
1. 检查网络连接
2. 稍后重试
3. 检查账户权限

### 导出失败

**可能原因**：
1. 磁盘空间不足
2. 文件权限问题
3. 数据过多

**解决方案**：
1. 检查磁盘空间
2. 检查文件权限
3. 分段导出
