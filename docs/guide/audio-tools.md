# 音频目录工具

音频目录工具提供了一系列音频管理和处理功能，帮助你高效地管理直播音频文件。

## 功能说明

### 工具包含

1. **音频格式转换** - 转换音频格式
2. **音频文件整理** - 批量整理音频
3. **前缀检查** - 检查音频前缀
4. **音频验证** - 验证音频文件完整性

---

## 音频格式转换

### 支持的格式

**输入格式**：
- MP3、WAV、AAC、M4A、FLAC、OGG

**输出格式**：
- MP3（推荐）
- WAV
- AAC

### 转换步骤

1. 进入 **音频目录工具** 页面
2. 点击 **格式转换** 标签
3. 选择要转换的音频文件
4. 选择目标格式
5. 点击 **转换** 按钮
6. 等待转换完成

### 转换建议

- **推荐格式** - MP3（兼容性最好）
- **高保真** - WAV（文件较大）
- **苹果设备** - AAC（iTunes格式）

## 音频文件整理

### 功能说明

批量整理音频文件，按照指定规则重命名或移动。

### 整理步骤

1. 进入 **音频目录工具** 页面
2. 点击 **文件整理** 标签
3. 选择源目录
4. 选择整理规则
5. 点击 **整理** 按钮
6. 确认结果

### 整理规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 按日期 | 按文件修改日期分类 | 2024-01/、2024-02/ |
| 按格式 | 按文件格式分类 | mp3/、wav/、aac/ |
| 按大小 | 按文件大小分类 | 小于1MB/、1-10MB/ |

## 前缀检查

### 功能说明

检查关键词分类与音频文件夹的对应关系，发现缺失或多余的音频。

### 检查步骤

1. 进入 **音频目录工具** 页面
2. 点击 **前缀检查** 标签
3. 点击 **检查** 按钮
4. 查看检查结果

### 检查结果示例

```
✅ 检查完成

以下分类缺少对应音频：
- 售后
- 退货

检测到新音频前缀（关键词未配置）：
- 新产品
- 限时优惠

建议：
1. 为"售后"和"退货"分类添加音频文件
2. 为"新产品"和"限时优惠"创建关键词分类
```

### 如何处理检查结果？

**缺失音频**：
1. 进入 **主播设置**
2. 找到对应的分类文件夹
3. 添加音频文件

**多余音频**：
1. 进入 **关键词设置**
2. 创建对应的关键词分类
3. 或删除多余的音频文件

## 音频验证

### 功能说明

验证音频文件的完整性和有效性。

### 验证步骤

1. 进入 **音频目录工具** 页面
2. 点击 **音频验证** 标签
3. 选择要验证的音频文件
4. 点击 **验证** 按钮
5. 查看验证结果

### 验证项目

| 项目 | 说明 |
|------|------|
| 文件存在 | 文件是否存在 |
| 文件大小 | 文件大小是否合理 |
| 音频格式 | 格式是否支持 |
| 音频时长 | 时长是否正常 |
| 音频质量 | 比特率、采样率等 |

## 常见问题

### Q: 如何批量转换音频格式？

A: 
1. 进入 **音频目录工具**
2. 点击 **格式转换** 标签
3. 选择源目录
4. 选择目标格式
5. 点击 **批量转换**

### Q: 转换后的文件在哪里？

A: 
1. 默认保存在源目录的 `converted/` 子文件夹
2. 或你指定的输出目录

### Q: 转换会删除原文件吗？

A: 不会。原文件保留，转换后的文件保存在新位置。

### Q: 如何快速找到缺失的音频？

A: 
1. 进入 **音频目录工具**
2. 点击 **前缀检查** 标签
3. 点击 **检查** 按钮
4. 查看"缺失音频"列表

### Q: 可以自定义整理规则吗？

A: 目前支持预设规则。如需自定义，可以手动整理或联系技术支持。

## 最佳实践

### 1. 定期检查音频

- 每周运行一次前缀检查
- 发现问题及时处理
- 保持音频库整洁

### 2. 统一音频格式

- 推荐使用MP3格式
- 统一音频质量
- 便于管理和播放

### 3. 备份重要音频

- 定期备份 `zhubo_audio/` 目录
- 避免音频丢失
- 建议每周备份一次

### 4. 清理过期音频

- 定期删除不用的音频
- 保持目录整洁
- 减少系统负担

## 故障排查

### 转换失败

**可能原因**：
1. 源文件损坏
2. 磁盘空间不足
3. 格式不支持

**解决方案**：
1. 检查源文件
2. 清理磁盘空间
3. 尝试其他格式

### 检查结果不准确

**可能原因**：
1. 关键词配置不完整
2. 音频文件夹名称不匹配
3. 缓存未更新

**解决方案**：
1. 检查关键词配置
2. 确认文件夹名称
3. 重新运行检查

### 验证失败

**可能原因**：
1. 文件损坏
2. 格式不支持
3. 文件权限问题

**解决方案**：
1. 尝试用其他播放器打开
2. 转换为支持的格式
3. 检查文件权限
