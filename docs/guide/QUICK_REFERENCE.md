# 快速参考卡

这是一份快速参考指南，包含所有常用操作和快捷方式。

## 🎯 核心操作

### 启动系统
```
AI工作台 → 🚀 启动系统 → 扫码登录 → 进入直播控制台
```

### 配置关键词
```
关键词设置 → 新建分类 → 配置必含词/意图词/排除词 → 保存并热更新

```
AI工作台 → 打开对应开关 → 功能启用

## 🎵 音频格式

### 支持的格式
- ✅ MP3（推荐）
- ✅ WAV（高保真）
- ✅ AAC（苹果格式）
- ✅ M4A（iTunes格式）
- ✅ FLAC（无损格式）
- ✅ OGG（开源格式）

### 推荐设置
- **格式**：MP3
- **比特率**：128-320 kbps
- **采样率**：44.1 kHz 或 48 kHz
- **时长**：5-30秒（单条）

## 🚨 故障快速排查

### 系统启动失败
```
1. 检查网络连接
2. 检查登录状态
3. 点击 🔄 重新运行
4. 查看日志错误信息
```

### 关键词没有触发
```
1. 确认 弹幕监听 开关已打开
2. 使用本地弹幕测试
3. 检查关键词配置
4. 查看日志中的匹配结果
```

### 音频没有声音
```
1. 检查系统音量
2. 点击 试听 按钮测试
3. 检查音频文件是否存在
4. 尝试转换为MP3格式
```

### 配置保存失败
```
1. 检查文件权限
2. 检查磁盘空间
3. 重启软件
4. 查看日志错误信息
```
